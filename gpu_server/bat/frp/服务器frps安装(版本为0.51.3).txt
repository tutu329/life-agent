linux下：
    下载frp：
        安装aria2c：
            sudo mkdir -p /etc/systemd/system/snapd.service.d
            sudo vi /etc/systemd/system/snapd.service.d/snap_tuning.conf，放入下述内容
[Service]
Environment="SNAPD_STORE=mirrors.tuna.tsinghua.edu.cn"
            sudo systemctl daemon-reload
            sudo systemctl restart snapd
            snap install aria2c
        多线程下载：
            aria2c -x 16 -s 16 https://github.com/fatedier/frp/releases/download/v0.51.3/frp_0.51.3_linux_amd64.tar.gz
    解压：
        tar -xzvf frp_0.51.3_linux_amd64.tar.gz
    配置frps.ini
[common]
bind_port = 7880
    运行：
        nohup ./frps -c frps.ini &
    切记：
        服务器要开7880端口和frpc.ini中所有端口
        然后client重启frpc：
            sudo systemctl start frpc.service（要确认域名确实切换到新的ip后，可以在client上通过ping powerai.cc来判断ip是否切换了，然后再启动frpc服务）

    ubuntu服务器配置frps自启动：
        sudo vi /etc/systemd/system/frps.service

[Unit]
Description=frps Service
After=network.target
Wants=network.target

[Service]
Type=simple
User=root
Restart=always
RestartSec=5s
ExecStart=/root/frp/frps -c /root/frp/frps.ini
ExecReload=/root/frp/frps -c /root/frp/frps.ini

[Install]
WantedBy=multi-user.target

        sudo systemctl daemon-reload
        sudo systemctl start frps.service
        sudo systemctl status frps.service
        sudo systemctl enable frps.service
